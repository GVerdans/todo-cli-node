## Todo-CLI

### This is a todo App to Linux Users. Run in terminal, and easly to use.

## To local run follow steps bellow:

### Clone this repo:

`git clone https://github.com/GVerdans/todo-cli-node.git`

```bash
cd todo-cli-node
npm install
npm link

```

> So run `todo` on terminal to use.
